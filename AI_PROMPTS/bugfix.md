# Bugfix Prompt

## Problem
Describe the visible failure, error message, or incorrect behavior.

## Expected Behavior
Describe what should happen instead.

## Reproduction
1. Open/run:
2. Action:
3. Result:

## Constraints
- Keep the fix focused.
- Preserve existing public behavior unless it is the bug.
- Add or update tests when the risk justifies it.

## Ask
Diagnose the root cause, implement the smallest durable fix, and verify it.
